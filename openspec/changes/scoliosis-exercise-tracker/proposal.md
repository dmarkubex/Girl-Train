## Why

每天需要手动用手机计时器跟踪女儿的脊柱侧弯康复锻炼，涉及多个项目、多组计时和休息间隔，操作繁琐且容易出错。需要一个专用的网页应用来自动化计时流程、记录锻炼数据，并通过可视化统计促进锻炼坚持和效果提升。

## What Changes

- 新建一个移动端优先的网页应用（PWA），用于脊柱侧弯康复锻炼跟踪
- 支持自定义锻炼项目配置（项目名称、每组时长、休息时长、组数、项目间休息时长）
- 内置自动计时器，按配置依次执行锻炼/休息/项目切换，带语音或声音提示
- 锻炼完成后生成当日统计图表（完成率、各项目用时、总时长等）
- 历史记录存储与对比，展示锻炼趋势和坚持情况
- 支持离线使用（PWA + 本地存储）

## Capabilities

### New Capabilities
- `exercise-config`: 锻炼项目配置管理 — 定义和编辑锻炼计划（项目列表、时长、组数、休息间隔）
- `workout-timer`: 锻炼计时引擎 — 自动按计划执行计时，管理锻炼/休息/项目切换状态，提供声音提示
- `session-tracking`: 锻炼记录追踪 — 记录每次锻炼的执行详情（实际完成组数、跳过情况、总时长等）
- `stats-dashboard`: 统计仪表盘 — 当日锻炼完成后的统计图表展示，及历史数据对比趋势图
- `data-persistence`: 数据持久化 — 使用本地存储（IndexedDB/localStorage）保存配置和历史记录

### Modified Capabilities

（无 — 这是全新项目）

## Impact

- 新建完整的前端项目（HTML/CSS/JS 或轻量框架）
- 需要 Web Audio API 或音频文件用于计时提示音
- 需要图表库（如 Chart.js）用于统计可视化
- PWA manifest 和 Service Worker 用于离线支持
- 本地存储 API（IndexedDB）用于数据持久化
- 无后端依赖，纯前端应用
